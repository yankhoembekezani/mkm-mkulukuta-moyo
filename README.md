# MKM — Mkulukuta Moyo

## Project Story
MKM (Mkulukuta Moyo) is a backend platform built to solve transparency, tracking, and audit issues in group-based contributions and collective investing.

## Goals
- Build a production-grade FastAPI backend with real-world financial workflows
- Enforce integrity rules (immutable ledger, approvals, audit logs)
- Implement secure multi-tenant access (organizations, roles, isolation)
- Ship a deployable, testable system with clean documentation
